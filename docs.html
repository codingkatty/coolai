QuickStart

Get started with CoolAI API in minutes.

1. Get Your API Key

  1. Sign up for an account at CoolAI Dashboard
  2. Verify your email address
  3. Log in to your account
  4. Your API key will be displayed and ready to use

2. Make Your First Request

const response = await fetch('https://coolai.onrender.com/generate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: 'What is artificial intelligence?'
  })
});

const data = await response.json();
console.log(data.response);

Authentication

All API requests require authentication using your API key.

Using API Keys

Include your API key in the Authorization header of all requests:

Authorization: Bearer YOUR_API_KEY

⚠️ Security Notice

Never share your API key or commit it to version control. Use environment variables to store sensitive information.

Generate Endpoint

Generate AI responses using the Gemini model.

Endpoint Details

POST https://coolai.onrender.com/generate

Headers

{
  "Authorization": "Bearer YOUR_API_KEY",
  "Content-Type": "application/json"
}

Request Body

{
  "prompt": "Your prompt text here"
}

Success Response

{
  "success": true,
  "response": "AI generated response text"
}

Error Response

{
  "error": "Error message",
  "details": "Detailed error information"
}

Notes

  • Uses Gemini 1.5 Flash model for fast response generation
  • Prompt text is required in the request body
  • Rate limits may apply to API usage

Examples

JavaScript

async function generateResponse(prompt) {
    const response = await fetch('https://coolai.onrender.com/generate', {
        method: 'POST',
        headers: {
            'Authorization': 'Bearer YOUR_API_KEY',
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({ prompt })
    });
    
    const data = await response.json();
    return data.response;
}

Python

import requests

def generate_response(prompt):
    response = requests.post(
        'https://coolai.onrender.com/generate',
        headers={
            'Authorization': 'Bearer YOUR_API_KEY',
            'Content-Type': 'application/json'
        },
        json={'prompt': prompt}
    )
    return response.json()['response']
